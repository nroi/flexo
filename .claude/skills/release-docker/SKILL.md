# Release a new version on Docker Hub

The Docker image is published at https://hub.docker.com/repository/docker/nroi/flexo.
The source repo is https://github.com/docker-flexo/docker-flexo — check it out locally before proceeding.

## Step 1 — Bump the version in flexo (if not done already)

Follow the `release-aur` skill.

## Step 2 — Update Docker Image

Follow instructions in https://github.com/docker-flexo/docker-flexo/blob/master/README.md#pushing-a-new-release
